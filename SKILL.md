# OpenClaw 豆瓣高分监控

## 目标
这个 skill 用来监控豆瓣里“近期新出现并达到门槛”的影视内容，默认关注以下条件：

- 豆瓣评分大于 `8.0`
- 豆瓣评分人数大于 `3000`
- 条目是首次达标，或者后续热度显著增长，值得进一步关注

## 模式
本 skill 支持两种运行模式，通过 `config.toml` 中的 `mode` 字段切换。

### 轻量模式（`mode = "lite"`，默认）

- 通过豆瓣 Frodo API（移动端/小程序内部接口）获取榜单候选和详情数据
- 不需要浏览器环境，不依赖 Playwright
- 适合无图形界面的服务器、容器和 CI 环境

### 完整模式（`mode = "full"`）

- 豆瓣榜单页使用浏览器抓取
- 豆瓣详情页优先使用 HTML 解析，失败时回退浏览器
- 宿主环境需要具备浏览器自动化依赖（Playwright + Chromium）

## 数据源策略
采用”豆瓣侧 + TMDB 侧”的混合候选池。

来源职责：

- 豆瓣榜单
  负责发现近期热门候选。轻量模式通过 Frodo API 获取，完整模式使用浏览器抓取
- 豆瓣详情
  负责提供评分和评分人数真值。轻量模式通过 Frodo API 获取，完整模式使用 HTML 解析
- TMDB 热门接口
  负责提供额外候选和展示元数据

默认配置的豆瓣榜单包括：

- 电影
- 华语剧集
- 全球剧集
- 国内综艺
- 国外综艺

## 抓取策略

### 轻量模式

1. 调用 Frodo API 获取豆瓣各榜单的候选列表，接口直接返回条目 ID、标题、评分和评分人数
2. 对于榜单已包含完整评分数据的条目，跳过详情页请求
3. 对于缺少评分数据的条目（如 TMDB 候选），调用 Frodo API 详情接口补全

### 完整模式

1. 使用浏览器打开豆瓣榜单页
2. 优先按正文文本中的”排名 -> 标题 -> 评分”结构提取候选标题
3. 如果榜单卡片可点击，则直接点击卡片补全详情页链接
4. 对详情页优先使用普通 HTML 提取标题、评分、评分人数和年份
5. 如果详情页普通请求缺字段，则回退到浏览器抓取详情页

## 执行流程

对应 `monitor.py` 的 `[1/7]` ~ `[7/7]`：

1. 抓取豆瓣多榜单候选
2. 抓取 TMDB 候选
3. 去重并补全详情页，提取评分和评分人数
4. 更新状态与监控库，判定新增命中、继续观察和二次提醒
5. 写入状态文件、监控库和 Markdown 报告
6. 生成网页数据：调用 `fetch_posters.py` 获取封面、`fetch_metadata.py` 获取元数据
7. 提交数据变更并推送到 GitHub

## 可视化网页

项目根目录的 `index.html` 是一个静态网页，读取 `data/` 下的 JSON 文件展示达标内容。

- 暗色主题瀑布流卡片布局
- 支持电影 / 剧集 / 综艺分类切换和多维排序
- 卡片包含封面、评分、类型标签、简介、上映日期，点击跳转豆瓣详情页
- 数据来源：`douban-monitor-result.json`、`douban-monitor-posters.json`、`douban-monitor-metadata.json`

## 核心规则

### 首次达标
满足以下条件时视为“新增命中”：

- 评分大于 `min_rating`
- 评分人数大于 `min_rating_count`
- 历史状态中此前未达标

### 继续观察
未达到最终门槛但仍有潜力的条目继续进入监控库，例如：

- 评分已经接近目标
- 评分人数已具备增长空间
- 仍然出现在榜单来源中

## 入库规则
不要把所有候选都永久加入监控库。只有具备一定潜力的条目才会继续跟踪。

入库参考条件：

- 出现在豆瓣榜单页
- 出现在选定的 TMDB 热门源
- 豆瓣评分大于等于 `7.5`
- 豆瓣评分人数大于等于 `1000`

## 当前状态

当前版本已经验证通过：

- 轻量模式：通过 Frodo API 获取榜单候选和详情数据，无需浏览器
- 完整模式：浏览器抓取豆瓣榜单候选、点击卡片补链接、详情页提取评分
- 状态文件、监控库、Markdown 报告输出
- 阈值筛选与新增命中判定
- 网页可视化展示（瀑布流卡片布局、封面、简介、评分）
- TMDB 封面和元数据自动获取
- GitHub Actions 定时自动运行（每天 09:00 / 21:00）

当前已打通的榜单类型：

- 电影
- 华语剧集
- 全球剧集
- 国内综艺
- 国外综艺

## 已知限制

- 轻量模式依赖豆瓣 Frodo API，如果接口变更或限流需要关注
- 完整模式下少量详情页在特定环境下可能无法稳定提取评分或评分人数
- 未配置 `TMDB_API_KEY` 时，TMDB 候选源和网页封面/元数据不会生效
- 未配置豆瓣 Cookie 时，不依赖豆瓣搜索页补链接
